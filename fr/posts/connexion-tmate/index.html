Se connecter à un autre ordinateur avec tmate

03-10-2022

2 minutes

 Twitter

 LinkedIn

Il y a une façon très simple d'accéder à un autre ordinateur via le terminal, et c'est avec le programme tmate.

J'explique ça ici très simplement.

D'abord, installer tmate.

Ouvrir un terminal et,

Pour Ubuntu
sudo apt-get install tmate
Pour Archlinux
pacman -S tmate
Pour NixOS
nix-env -iA nixos.tmate

Lancer la commande tmate dans un terminal.

Elle produira le message d'introduction suivant.

Tip: if you wish to use tmate only for remote access, run: tmate -F                                                                                       [0/0]
To see the following messages again, run in a tmate session: tmate show-messages
Press <q> or <ctrl-c> to continue
---------------------------------------------------------------------
Connecting to ssh.tmate.io...
Note: clear your terminal before sharing readonly access
web session read only: https://tmate.io/t/ro-4VefBGwwwUvNMV4fLQjsHFZ2d
ssh session read only: ssh ro-4VefBGwwwUvNMV4fLQjsHFZ2d@sgp1.tmate.io
web session: https://tmate.io/t/GdkathWe7kqqu9JVQgt2KFqrt
ssh session: ssh GdkathWe7kqqu9JVQgt2KFqrt@sgp1.tmate.io

Cette page donne les informations de connexion de cette session tmate.

Sur un autre ordinateur, on peut accéder à la session tmate soit…

  • en navigant à l'url suivant web session read only, pour un accès en tant que spectateur
  • en navigant à l'url suivant web session, pour un accès complet, nous permettant d'écrire des commandes
  • en utilisant un terminal, et avec une des deux commandes ssh données (pareil, read only pour être spectateur, sinon l'autre)
  • Enfin, il suffit d'appuyer sur Q pour quitter le message d'introduction et de d'avoir le prompteur.
